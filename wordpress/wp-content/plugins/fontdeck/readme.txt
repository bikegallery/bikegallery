The official Fontdeck plugin. Add Fontdeck web fonts to your WordPress site.

== Description ==

The WordPress plugin makes it easy to add custom fonts to your website.

Once you've chosen your fonts on the [Fontdeck](http://fontdeck.com) website, you can add them to your Wordpress website using this plugin.

== Installation ==

1. Upload the folder 'fontdeck' and it's contents to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a website on [Fontdeck](http://fontdeck.com) and add your fonts
4. Paste the project ID from Fontdeck into the setting page.
5. Choose which types of text are targeted by which font.

In advanced mode, you can specify the selectors for each font.

== Frequently Asked Questions ==

= Where do I go to get the Project ID? =

Once you have created a website on [Fontdeck](http://fontdeck.com), go to your account, click on the website and copy the Project ID you see at the top of that page.

= Can I add this plugin before I'm registered on Fontdeck? =

Yes. This plugin won't do anything until you paste the code in from Fontdeck. Feel free to install the plugin first and then paste the code when you are ready.
